Show Source Content Sections In Templates

Overview

Adding a `-- template`` section to a page provides a way to pass variables to to the rendering process. You can show the source of content sections by setting this above them:

-- template
-- show_source: on

Note

This functionality is provided by the theme templates. It's built into the default Neopoligen theme but may not be available in others if the authors don't include it.

Here's an example of what it looks like when you turn it on:

Turning it off is done with:

-- show_source: off
