Helpful Error Messages

Note

Dev is still in the prototype phase. Error messages are often super cryptic. This post doesn't describe the way things are. It's the direction to march towards

The rust compiler is amazing once you learn how to read it.

It often walks you to the point that you need to fix and frequently provides an suggestion that solves the problem.

That's the goal. If someone can't figure out what to do when there's an error by reading the error message it's a bug.
