Debugging Controls

Neopoligen is Free, Open-Source Software. That means anyone can download their own copy of the app's code to play with or even make their own version of the app. This page is for features to help support that.

Source Code Repository

Delete Neopoligen Config File

This button deletes Neopoligen's main config file. Clicking it and then restarting the app will sets the 'Active Site' to the 'Example Site' and will rebuild it if it doesn't already exist. It should not affect any of your sites, but it's always best to make sure they are backed up first.

Check Version Number

Neopoligen doesn't yet automatically check for updates. You're running version 0.0.1. You can check the latest available version here:

https://github.com/neopoligen/neopoligen/blob/main/version.json
