Content Files

Neopoligen keeps your original content files directly on your computer. They're in a "Neopoligen" folder under your main 'Documents' folder. The app can make multiple sites so each one gets its own folder. When you first install the app, it makes a new site in the 'Example Site' folder which you can see here:

The initial contents of a site folder are:

  • config.json file - Contains some basic details about the site. (Details on the configuration can be found in the documentation)
  • configuration folder - This also holds some configuration items, but it's going away. Everything is moving into the config.json file
  • content folder - This is where all the main content files that make up your site go. The files all end with a .neo extension (if you can see those in your finder) which is the type of file that Neopoligen uses.
  • docs folder - This is where the files that make up the actual website are output when Neopoligen builds them. You don't need to mess with it, but feel free to poke around to see what the output is.
  • images folder - For any images you want to include on your site. You can organize images however you want inside this folder and then include them in on a page using just the name of the file. More details: images folder documentation)
  • themes folder - Neopoligen works by combining your content files with a theme to build the site. This is where they are stored.
